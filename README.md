# otpfx

<img src=screenshort.png>

An otp UI written using javafx, supports TOTP and HOTP.

# Prerequisit
- openjdk 8
- openjfx

To get the pre-requisits
```
sudo dnf install java-1.8.0-openjdk java-1.8.0-openjdk-openjfx
```

# Run it
```
java -jar otpfx.jar
```

# Compile it
```
mvn clean package
```
